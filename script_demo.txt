## 🏗️ Technical Architecture Overview (For Reference)
- **Frontend**: Next.js (App Router), React Hooks, Vanilla CSS (Neo-Brutalist design).
- **Backend**: Python Flask REST API.
- **AI**: Google Gemini 1.5 Flash (Product identification & curated deal engine).
- **Data**: SerpApi (Google Shopping real-time aggregate) + MySQL (Authentication & Persistence).
- **State**: React Context API (`SearchContext`) + Browser LocalStorage.

## Part 2: Core Feature - Visual Discovery & AI Identification
**Speaker: Teammate 2**
*Slide/Screen: Search Page -> Image Upload/Webcam*

"The heart of ShopSmart is its **Visual Discovery Engine**. Most platforms force you to know the exact name of what you want; we let you search with your eyes.

(Demonstrate Image Upload/Capture)
Through our `/api/identify` endpoint, we leverage **Google Gemini 1.5 Flash**. As soon as you upload an image, our AI performs an immediate multi-point analysis, returning a structured JSON object containing the product name, brand, model, and category in milliseconds.

But here’s where we differentiate: **Parallel Orchestration.** Instead of waiting for one provider, we use asynchronous `Promise.all` calls to hit multiple deal sources at once. This includes our custom **Gemini Deal Engine** and the **SerpApi Google Shopping Aggregate**, ensuring you get the most comprehensive price-map possible."

## Part 3: Premium Enhancements & Performance Logic
**Speaker: Teammate 3**
*Slide/Screen: Results Dashboard (Toggling between Gemini and Shopping)*

"Efficiency is nothing without an elite User Experience. We’ve implemented several **Premium Enhancements** that set ShopSmart apart.

(Switch between Option 1 and Option 2)
1. **Instant Engine Switching**: Because we pre-load all results in parallel, you can swap between AI-curated deals and raw Shopping results instantly. There’s no secondary loading time—the data is already there.
2. **Deep-Blur Loading System**: Notice our full-page loading backdrop. It uses CSS `backdrop-filter` and dynamic status messaging to keep users engaged during the scan. 
3. **Intelligent Fallbacks**: AI results often lack images. We’ve built a 'Smart-Thumb' logic: if a result is missing a thumbnail, we automatically fallback to your original uploaded photo, keeping the dashboard polished.
4. **Load More Pagination**: To maintain a 'light' DOM and fast performance, we display 6 elite results initially, with a bold pagination system to expand the list as needed."
