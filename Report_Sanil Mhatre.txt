1. Project Title
ShopSmart: Web Application for Product Price Comparison Across E-Commerce Sites

2. Project Objective
The main goal of the ShopSmart project is to alleviate the inefficiency and time-consuming nature of deal hunting across fragmented online stores. The project aims to provide an AI-powered price comparison application that automates product identification via images and text, helping users find the lowest market prices in seconds. This relevance to the organization lies in enhancing consumer decision-making and optimizing the e-commerce shopping experience.

3. Project description in detail
ShopSmart is an evolved, full-stack e-commerce solution designed to bridge the gap in market price transparency by transforming a core script into a scalable web application.

Approach: 
The project follows a modular, visual-first workflow. It starts with an AI-driven recognition engine that identifies products from user-uploaded images or text queries. This is followed by a robust scraping engine using SerpApi and a sophisticated filtering system to ensure "exact matches" and exclude irrelevant items. The backend coordinates data between the scraping logic and the frontend, which uses a modern Neo-Brutalist design.

Technology Used:
- Frontend: Next.js (Modern React Framework), Neo-Brutalist design, Vanilla CSS.
- Backend: Python Flask (Robust API Proxy), pymongo (NoSQL integration).
- Database: MongoDB Atlas (Cloud-hosted cluster), transitioned from MySQL for scalability.
- AI Engine: Google Gemini AI (Product Recognition & Intelligent Scraping).
- APIs: SerpApi, BeautifulSoup (for real-time price aggregation).

Impact: 
ShopSmart empowers consumers with actionable market data, significantly reducing the time spent on manual price comparisons—moving from "Tab-Hopping" to instant saving. Its real-world implementation prioritizes efficiency, visual ease, and user-centric savings.

5b. Project execution details
The project was executed in four key phases:
1. Core Scraping Engine: Developing a robust Python scraper with strict error handling and advanced filtering to exclude accessories and irrelevant parts.
2. AI Visual Search: Building a structure for image-recognition tools that simulate AI responses for precise product identification.
3. UI/UX Design & Copywriting: Crafting a unique Neo-Brutalist aesthetic and benefit-driven copy like "Stop Tab-Hopping. Start Saving."
4. Custom Branding: Creating visual assets including a custom logo with the ₹ symbol and playful sticker assets of internet culture icons for a modern feel.
5. Database Optimization: Migrating from a relational structure (MySQL) to a flexible, NoSQL document-based structure (MongoDB) to handle dynamic product schemas efficiently.
